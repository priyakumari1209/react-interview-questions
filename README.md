- **React Basics**: JSX, Props, State, Events

- **State Management**: Redux, Context API, Zustand  
- **Data Fetching**: React Query, Axios patterns  
- **Performance Optimization**: memoization, virtualized lists  
- **Testing**: Jest, React Testing Library  
- **TypeScript**: React + TS best practices  
- **Architecture Patterns**: Component patterns, Atomic design

- **Advanced Concepts**: Render Props, HOCs, Portals, Suspense, Concurrent Mode
